Contexto

Este mapa foi criado como parte do cluster de energia fora da rede (off-grid) da GIZ em Moçambique. Destina-se a apoiar o desenvolvimento do setor off-grid, proporcionando melhor acesso a dados do mercado, para promotores, distribuidores e investidores de mini-redes e sistemas domésticos de energia solar.

O mapa mostra dados para todas as províncias de Moçambique, usando apenas dados disponíveis publicamente. Ele incorpora análises combinando dados públicos e de satélite para ajudar os promotores a identificar mercados e clientes potenciais.

Fonte Aberta

A lista completa do conjuntos de dados públicos usados e os métodos utilizados para tratá-los e analisá-los está disponível no repositório GitHub do projeto moz-offgrid-data.

O código e os recursos usados neste site estão disponíveis num repositório separado moz-offgrid-viz.
